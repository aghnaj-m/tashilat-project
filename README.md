# tashilat-project

this application aims to approach consumer payment services, it mainly includes 4 services:

<li>Electricity bills payment ;</li>
<li>Water bills payment ;</li>
<li>Phone top up ;</li>
<li>Money transfer ;</li>

## Dev Tools:

<li>Spring Boot</li>
<li>Angular</li>
<li>JWT</li>
<li>MySQL database</li>
